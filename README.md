# POC – Food & Beverage Website

Live Site:
https://pocfnbsite.developershideout.com/

Tech Stack:
- WordPress
- Elementor
- JavaScript
- CSS
- Adobe XD

Overview:
Proof-of-concept website demonstrating layout and UX design for food and beverage businesses.

Purpose:
Built to showcase how restaurants or food brands can present menus, branding, and promotions through a structured and responsive layout.

Features:
- Hero section with branding focus
- Menu layout sections
- Call-to-action blocks
- Responsive mobile design

What I Focused On:
- Visual hierarchy
- Spacing and layout balance
- Mobile-first responsiveness
